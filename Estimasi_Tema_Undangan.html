Estimasi & Analisis Biaya

Pengembangan Tema Undangan Digital

Proposal by: Bhaktiaji Ilham
Tanggal Dokumen: 17 Mei 2026
Kerahasiaan: Internal / Klien Pak Akrom17

Berdasarkan hasil audit teknis pada source code infrastruktur sistem undangan-digital-groovy, proses pembuatan tema baru sangat dipengaruhi oleh desain arsitektur database dan struktur komponen antarmuka (Frontend React). Untuk memudahkan estimasi bagi klien, pembuatan layanan tema dibagi menjadi tiga (3) tingkatan klasifikasi teknis (Tiers) sebagai panduan *order* sistematis.

Tier 1: Basic Theme (Minimalis)

Implementasi menggunakan layout default dengan penyesuaian khusus pada palet warna dan tipografi.

  • Infrastruktur Teknis: Tidak membutuhkan modifikasi source code. Bekerja secara otomatis melalui 1 baris konfigurasi JSON pada skema tabel themes dan theme_sections.
  • Skup Modifikasi: Penentuan Hex Code Color & Integrasi font melalui CDN (Google Fonts).
  • Tingkat Kesulitan: Sangat Mudah ⭐
Estimasi Waktu Integrasi (per tema) ± 10 Menit
Harga Wajar (per tema) Rp 15.000
Penawaran Bundling Paket (5 Tema) Rp 70.000

Tier 2: Premium Theme (Custom Decoration)

Penyesuaian tata letak lanjut menggunakan injeksi ornamen grafis spesifik (seperti motif Adat Jawa, Sunda, dll).

  • Infrastruktur Teknis: Memerlukan injeksi tata letak aset grafis transparan (.png/.webp).
  • Modifikasi DOM: Membutuhkan manipulasi statis pada source code frontend di dalam file induk pendakian resources/js/Pages/Invitation/Show.jsx untuk mengondisikan logic ornamen tambahan di setiap titik scroll (mobile/desktop).
  • Tingkat Kesulitan: Menengah ⭐⭐⭐
Estimasi Waktu Integrasi (per tema) ±1 - 2 Jam
Harga Wajar (per tema)* Rp 50.000
Penawaran Bundling Paket (5 Tema) Rp 235.000

*Tidak termasuk biaya pembuatan IP (Intellectual Property) jika dilakukan ilustrasi kustom dari 0.

Tier 3: Exclusive Theme (Custom Layout Total)

Rancang bangun tema berbasis independensi tata letak secara 100%, lepas dari batasan tema reguler.

  • Infrastruktur Teknis: Restrukturisasi DOM, inisasialisasi script khusus (Parallax, Advanced Transitions, dll).
  • Modifikasi Root: Pembuatan Page rendering baru secara utuh terpisah dari layout master (contoh pola referensi: HeritageShow.jsx).
  • Konfigurasi Backend: Merombak arsitektur routing theme di dalam InvitationController.php berbasis slug identifier.
  • Tingkat Kesulitan: Sangat Tinggi ⭐⭐⭐⭐⭐
Estimasi Waktu Integrasi (per tema) ±1 Hari Kerja
Harga Wajar (per tema) Rp 135.000
Penawaran Bundling Paket (5 Tema) Rp 650.000

Dokumen ini dirumuskan berdasar analisis sistem per 17 Mei 2026. Harga di atas adalah indikator kewajaran teknis dan dapat disesuaikan dengan nilai komersial operasional startup.
